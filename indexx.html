Introduction to Machine Language

We cant use human language to tell computer what to do, because computer doesnt know (or) understand human languages,it has its own language called machine language (or) binary language,so to communicate with computer (or) machine we have to learn machine language.

Machine Language(1st generation language):

Machine language is a set of instructions (or) commands, containing only 0's and 1's,which are directly executed bt computers (or) machines.

Ex-Code:-Machine Code for hypothetical machine:
